# step
Конструктор
```c++
step(STEP, DIR, MS1, MS2, MS3, 16 долей шага в единице)
```
Позиция в установленных единицах
```c++
pos
```
Движение
```c++
move(координата в установленных единицах (по умолчанию 0),
     координаты - 0 (абсолютные, по умолчанию)/1 (относительные),
     скорость - 0-255 (по умолчанию 128),
     микрошаг (по умолчанию 16))
```
